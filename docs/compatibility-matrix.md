# Compatibility Matrix

This document tracks parity between the Go implementation and the existing product behavior.

1. CLI flags and defaults
2. Config precedence
3. Scan and plan behavior
4. Upload retries and fallback paths
5. State and queue semantics
6. MCP HTTP and SSE behavior
